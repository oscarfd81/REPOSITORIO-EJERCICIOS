## DESCRIPCIÓN CONTENIDO

Este repositorio contiene los siguientes ejercicios, cada ejercicio está organizado en una carpeta:

* EJERCICIOS DE FORTY: Ejercicios sobre git en local y en remoto.
* EJERCICIOS DE LINUX:
  * MONTAJE MÁQUINA VIRTUAL: Explicación paso a paso del montaje de la máquina Ubuntu desktop en la que trabajaremos.
  * REPASO COMANDOS LINUX: Comandos básicos de repaso en la consola de Ubuntu. 
  * PERMISOS LINUX: Trabajar sobre atribuir permisos a usuarios, grupos y resto de miembros del sistema con chmod.
  * REDES LINUX: Configurar interfaces de red, enviar paquetes, configurar máscaras de subred...
